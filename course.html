Kursus Intensif

Program Kelas Kursus Web & IT

Belajar dari dasar sampai mahir dalam pengembangan website, database, dan tools digital. Program ini dirancang untuk pelajar, pemula, dan peserta yang ingin membangun karir teknologi dari Papua.

Online / Offline Sertifikat Resmi Mentor Lokal Project Nyata

Durasi

12 minggu pembelajaran intensif dengan sesi tatap muka dan praktik proyek setiap minggu.

Level

Mulai dari pemula, menengah, hingga persiapan karir junior developer.

Hasil

Membuat website responsif, backend sederhana, dan portofolio digital.

Komunitas

Bergabung ke komunitas EducasiCode untuk peluang magang, kerja, dan kolaborasi proyek.

Rangkaian Modul Kelas

Setiap modul dirancang agar peserta mudah mengikuti, dengan materi teori singkat dan praktik langsung pada proyek nyata.

Modul 2: JavaScript Interaktif

Memahami logika pemrograman, DOM, event, dan membuat fitur interaktif pada website.

Modul 3: Backend Sederhana

Dasar server, database, dan API untuk menyimpan data pengguna serta membuat form pendaftaran.

Modul 6: Portofolio

Menyusun website portofolio pribadi sebagai bukti hasil belajar dan representasi skill.

Detail Kursus

  • Jadwal: Senin & Kamis, 18.00 - 20.30 WIT
  • Durasi: 12 minggu / 36 sesi
  • Metode: Online + Offline Papua
  • Sertifikat: Sertifikat akhir kelas resmi

Apa yang Kamu Dapatkan

  • Project website portofolio
  • Bimbingan mentor dan review langsung
  • Akses komunitas EducasiCode
  • Persiapan kerja sebagai Junior Developer

Pertanyaan Umum

Apakah saya harus punya pengalaman coding?

Tidak, kelas ini dibuat untuk pemula. Semua peserta akan dibimbing dari dasar sampai dapat membangun project sendiri.

Apakah ada tugas proyek?

Ya, setiap modul memiliki tantangan praktis dan sesi akhir untuk membuat project portofolio.

Bagaimana saya mendaftar?

Tekan tombol "Daftar Sekarang" di halaman ini, lalu isi formulir pendaftaran dan tim kami akan menghubungi Anda.

Siap bergabung dengan kelas kursus kami?

Mulai pelatihan teknologi dengan mentor lokal, bangun portofolio, dan dapatkan dukungan komunitas untuk melangkah ke karir digital.
